# AI-Powered Multi-Market Trading System

Automated trading platform that monitors institutional investor behavior, cryptocurrency whale movements, and technical market signals across stocks, crypto, and forex markets.

## Features

- **Institutional Conviction Tracking**: Monitors 13F filings for stock market insights.
- **Crypto Whale Monitoring**: Tracks large cryptocurrency movements.
- **Technical Analysis**: Real-time signal generation using price and volume indicators.
- **AI Sentiment Analysis**: Leverages Claude API for market sentiment and reasoning.
- **Automated Execution**: Connects to Interactive Brokers, Alpaca, Kraken, and OANDA.
- **Risk Management**: Dynamic position sizing and strict drawdown controls.

## Tech Stack

- **Backend**: Python 3.10+, FastAPI, PostgreSQL, SQLAlchemy
- **Frontend**: React, Vite, Tailwind CSS
- **AI**: Claude API (Anthropic)
- **Data**: yfinance, ccxt, Whale Wisdom/SEC-API
